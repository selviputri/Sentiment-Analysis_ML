# Image-Based Sentiment Analysis

## Topic  
Image-Based Sentiment Analysis using Deep Learning

---

## Introduction  
This project focuses on analyzing sentiment from images using deep learning techniques. Unlike traditional sentiment analysis, which relies on textual data, this project investigates how effectively sentiment (positive, neutral, or negative) can be inferred purely from visual content.  

By using convolutional neural networks, the model learns visual patterns such as objects, scenes, and expressions that may convey emotional meaning. This allows us to explore the potential and limitations of image-only sentiment classification.

---

## Objective  
The main objectives of this project are:
- To build a deep learning model that classifies images into positive, neutral, or negative sentiment  
- To evaluate the model’s performance using standard metrics  
- To analyze how well sentiment can be captured from images alone  
- To identify cases where visual information is insufficient or ambiguous  

---

## Dataset

We use the MVSA-Single dataset for this project, which contains social media images labeled with sentiment (positive, neutral, negative).

Download the dataset from:
https://www.kaggle.com/datasets/vincemarcs/mvsasingle

After downloading, extract the dataset and place it in the following directory:

data/raw/
